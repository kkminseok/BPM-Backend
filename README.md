# 개요

- BPM(Body Profile Manager) 초기 버전 앱입니다.
- 현재 유지보수 및 추가 기능 구현을 위해 **private** 레포지토리로 옮겨서 작업중에 있습니다.
- 디자이너 2명, 안드로이드 개발자 2명, 백엔드 개발자 2명과 함께 만든 앱입니다.
